# WisprFlow

**Recommended speech-to-text tool for License Corporation developers.**

WisprFlow is a commercial voice dictation platform that uses LLM post-processing to produce clean, fluent text from speech.

## Why WisprFlow

| Feature | Benefit |
|---------|---------|
| **LLM post-processing** | Removes filler words, fixes errors, adds punctuation |
| **100+ languages** | Auto-detection and code-switching |
| **Personal dictionary** | Learns your terminology and proper nouns |
| **Cross-platform** | Mac, Windows, iPhone |
| **40+ app integrations** | Slack, Gmail, VS Code, Cursor, Notion, Claude |

## Setup

1. Download from [wisprflow.ai](https://wisprflow.ai)
2. Install and create account
3. **Decline training data usage** if desired (privacy option)
4. Default hotkey: **Ctrl+Win+Space**

## Usage

1. Press **Ctrl+Win+Space** in any text field
2. Speak naturally
3. Release when done
4. Clean, edited text appears

## Key Features

### Voice Commands
- Natural language commands for editing
- "Scratch that" to undo
- "New paragraph" for formatting

### Personal Dictionary
WisprFlow learns your vocabulary over time. Add terms like:
- Company names: License Corporation, licensecorporation-dev
- People: Jonathan Rose, Vic Garov
- Tech: Bazel, FedRAMP, Vertex AI, Claude Code

### Snippets
Create voice shortcuts for common text:
- "Insert meeting link" → your Zoom/Meet URL
- "Insert signature" → your email signature

### Tone Adjustment
Output adapts based on context:
- Slack → casual
- Email → professional
- Code comments → technical

## Developer Integration

Works with:
- **VS Code / Cursor** - dictate code, comments, commit messages
- **Terminal** - voice commands (be careful!)
- **Claude Code** - dictate prompts naturally
